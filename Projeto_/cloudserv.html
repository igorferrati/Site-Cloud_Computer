Amazon Web Services AWS

Plataforma de nuvem mais adotada e mais abrangente do mundo, oferecendo mais de 200 serviços completos de datacenters em todo o mundo. Milhões de clientes, incluindo as startups de crescimento mais rápido, grandes empresas e os maiores órgãos governamentais, estão usando a AWS para reduzirem seus custos, ficarem mais ágeis e inovarem mais rapidamente.

  • Domínio mercado cloud
  • Gama de ferramentas e recursos em crescimento
  • Serviços como:
    • Machine Learning do Amazon Aurora
    • Amazon Elasticache do qual turbina o desempenho de banco de dados
    • AWS Robo Maker para desenvolver, testar e implantar aplicativos inteligentes para robótica entre outros.

Google Cloud

O Google Cloud Platform foi desenvolvido com um objetivo em mente: favorecer a execução de uma série de serviços com base na computação em nuvem. Dessa forma, os empreendimentos podem contar com uma alta performance, segurança e confiabilidade em seus processos, otimizando a qualidade dos resultados adquiridos.

  • Experiência técnica profunda
  • Destaque em IA (Inteligência Artificial), machine learning e análise de dados
  • Grande crescimento e desenvolvimento de seus serviços
  • Serviços populares como:
    • Google Fotos
    • Google Drive
    • Google Docs
    • Entre outros

Microsoft Azure

Os serviços da Microsoft Azure permite a empresas e desenvolvedores tenham capacidade de processamento e armazenamento dos datacenters da Microsoft para aplicação em seus negócios como alternativa ao modelo convencional.

  • Forte no modelo SaaS
  • Serviços e integrações com aplicativos microsoft
  • Softwares locais adptados para nuvem como:
    • Windows Server
    • Office
    • SQL Server
    • .NET
  • Além de:
    • Kubernetes
    • Office 365
    • Inteligência Artificial
    • Banco de dados e etc

IBM Cloud

A IBM Cloud fornece soluções que permitem níveis mais altos de conformidade, segurança e gerenciamento, para entrega rápida e execução de cargas de trabalho de missão crítica.

  • Os serviços possuem características públicas, dedicadas, locais e hibridas
  • Serviços como:
    • IOT
    • APIs de Watson, podendo gerenciar, proteger e analisar os dados
    • IBM Watson Studio, construção e treinamento de modelos de inteligencia artificial

Sales Force

A Salesforce é uma empresa de softwares que foca na solução de gerenciamento de relacionamento para aproximar empresas e pessoas. É uma plataforma de CRM integrada que oferece a todos os departamentos uma visão única e compartilhada de cada cliente.

  • Seu software de CRM
    • Gerenciar e analisar as interações com clientes,
    • Relacionamento com o cliente
    • Armazena informações de clientes atuais e potenciais
    • Reúne e integra dados valiosos para preparar e atualizar suas equipes
    • Entre outras facilidades do CRM

Types of clouds

IaaS (Infrastructure as a Service)

  • Componentes básicos de TI na nuvem
  • Acesso a recursos de rede
  • Hardware dedicado
  • Espaço para armazenamento de dados
  • Alta flexibilidade e gerenciamento

PaaS (Platform as a Service)

  • Gerenciamento de recursos pelo provedor
  • Banco de dados
  • Filas
  • Mecanismos de fluxo de trabalho
  • Emails
  • Componente para criar aplicações

SaaS (Softwares as Service)

  • Aplicativo de recursos adiministrativos para o cliente
  • Capacidade de criação de aplicativos
  • Gerenciamento
